Flex Box, Listas y enlaces

Proposito del sitio


El proposito de este sitio es mostrar el uso de Flexbox con css, el uso de listas y de enlaces


Flexbox

Attardi J.(2020) define a Flexbox como un módulo de diseño de caja flexible, flexbox es su abreviación y la forma en como es mayormente conocido. Es una herramienta poderosa para crear diseños con CSS. No es tan poderoso como CSS Grid, pero puede resolver muchos problemas de diseño. Flexbox es un diseño unidimensional que puede colocar elementos horizontal o verticalmente (pero no ambos). Un elemento que usa flexbox como diseño se conoce como contenedor flexible, y los elementos dentro de él son elementos flexibles.


Listas

Según Prescolt(2015). Las listas se usan para agrupar información parecida o relacionada en un web, Permiten a los visitantes tener acceso a grupos de información de manera sencilla. En la lista se pueden incluir toda clase de elementos, desde enlaces hasta ennumercaiones de cosas.


Enlaces

Los enlaces, vínculos o hipervínculos, también conocidos como anclajes o links, son uno de los elementos más usados en HTML. Enlazan las páginas web. Permiten al usuario cargar una página externa en una red o acceder a otro lugar en la misma página donde se encuentra. También son usados en ocasiones para ejecutar funciones JavaScript. En este artículo explicamos cómo usarlos y todo lo que debemos saber, para hacer las páginas más funcionales y seguras Luján-Mora, S. (2013)
